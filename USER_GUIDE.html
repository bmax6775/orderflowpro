Pricing Plans Super Admin

Creating and managing subscription plans

üìã Default Plans

OrderDesk includes four standard plans:

  • Basic ($29/month): 3 agents, 500 orders
  • Professional ($59/month): 10 agents, 2000 orders
  • Enterprise ($99/month): Unlimited agents and orders
  • Custom: Fully customizable pricing and limits

‚ûï Creating Custom Plans

1. Click "Create New Plan" in Pricing Plans
2. Set plan name, price, and limits
3. Define features (comma-separated)
4. Mark as custom if special configuration needed
5. Save plan for assignment to users

‚úèÔ∏è Editing Plans

Existing plans can be modified, but changes affect all users on that plan. Consider creating new plans for different pricing.

üîÑ Order Status Workflow

üìù Updating Order Status

1. Click order ID or "Edit" button

üí¨ Adding Remarks

  • "Wrong address provided, getting correction"
  • "Customer confirmed order and payment"
  • "Delivery attempted, customer not home"

CSV Upload Guide Admin

Bulk order import instructions

üìã CSV Format Requirements

Required Columns (in order):
1. Order ID
2. Customer Name
3. Customer Phone
4. Customer City
5. Product Name
6. Product Price

‚úÖ CSV Best Practices

  • Use demo_orders.csv as template
  • Ensure unique Order IDs
  • Format phone numbers consistently
  • Use decimal format for prices (99.99)
  • Avoid special characters in names
  • Save as UTF-8 encoded CSV

‚ùå Common CSV Errors

Avoid these mistakes:
  • Missing required columns
  • Duplicate Order IDs
  • Empty rows in CSV
  • Incorrect price formatting
  • Special characters causing parsing errors
