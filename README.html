Overview

OrderDesk is a comprehensive eCommerce order management system designed specifically for Shopify and dropshipping businesses. It provides a complete solution for managing orders, tracking deliveries, monitoring agent performance, and handling customer communications - all through an intuitive web interface.

Perfect for: Small to medium businesses, dropshipping operations, call centers, and eCommerce teams who need efficient order management without complex integrations.

System Requirements

Minimum Requirements
  • PHP: Version 7.4 or higher
  • MySQL: Version 5.7 or higher (or MariaDB 10.2+)
  • Web Server: Apache or Nginx with mod_rewrite
  • Storage: 100MB disk space minimum
  • Memory: 128MB PHP memory limit (recommended: 256MB)
Recommended Environment
  • Hosting: Shared hosting (Namecheap, HostGator, etc.)
  • Control Panel: cPanel with phpMyAdmin access
  • SSL Certificate: For secure data transmission
  • Backup: Regular database and file backups

Configuration

Database Settings (config.php)
// Database configuration
$host = 'localhost';          // Database host
$dbname = 'orderdesk';        // Database name
$username = 'root';           // Database username
$password = '';               // Database password

// Environment variable support
$host = getenv('DB_HOST') ?: $host;
$dbname = getenv('DB_NAME') ?: $dbname;
$username = getenv('DB_USER') ?: $username;
$password = getenv('DB_PASS') ?: $password;
